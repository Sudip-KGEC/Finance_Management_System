# 💰 FinanceQ

A modern **Finance Dashboard** built with a clean UI, responsive design, and powerful features to manage transactions, track expenses, and visualize insights.

---

## 🚀 Features

### 📊 Dashboard

* Overview of **balance, income, and expenses**
* Interactive **charts (Recharts)**
* Clean card-based UI

### 💸 Transactions

* Add, delete, and filter transactions
* Search and category filtering
* Income vs Expense highlighting

### 📈 Insights

* Category-wise spending visualization
* Monthly income vs expense charts

### 🔐 Authentication

* Login system with validation
* Error handling & user feedback

### 🎨 UI/UX

* Modern **glassmorphism design**
* Light & Dark mode support 🌞🌙
* Smooth animations and transitions

### 📱 Responsive Design

* Fully optimized for:

  * Mobile 📱
  * Tablet 💻
  * Desktop 🖥️

---

## 🛠️ Tech Stack

### Frontend

* React.js
* SCSS (Modular & Clean Architecture)
* React Router
* Recharts (for charts)

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)


## 🌗 Theme Support

* Light Mode 🌞
* Dark Mode 🌙
* Smooth toggle with animated icons
